Brew it
-
This project is a fully functional web platform built to simulate a real-world coffee shop experience. Users can browse menu of premium brews,Juices and manage an interactive shopping cart. 
-
 Key Features
 ------------
Interactive Menu: Filterable categories (Hot Drinks, Cold Brews, Pastries) with live pricing.
Dynamic Shopping Cart:Real-time subtotal calculations and order persistence.
Responsive Design:can adjust size according to the screen size and will show a splash animation when navigating to certain pages.


 Tech Stack
 ----------
Frontend: HTML5, CSS3 (Flexbox/Grid), JavaScript
Backend: PHP 
Database: MySQL 
Design/UI: Inspired from Lovable Ai
